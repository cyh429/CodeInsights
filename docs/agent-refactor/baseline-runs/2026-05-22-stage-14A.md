# 2026-05-22 阶段 14A Agent Runner v2 默认化

## 结论

阶段 14A 已完成 Agent Runner v2 默认化：未设置 `RV_AGENT_RUNTIME_RUNNER_V2` 时默认走 `InProcessAgentRuntimeRunner`，显式设置 `RV_AGENT_RUNTIME_RUNNER_V2=0` 可回到旧 Agent 主循环，显式开启值继续强制 Runner v2。

本阶段没有默认开启 Pipeline Runner v2 或 Channels v2，也没有删除旧 Agent 主循环、Pipeline legacy adapter、旧 Feishu bridge 或旧 session JSONL 兼容。

版本：

- `@rv-insights/shared`: `0.1.40`
- `@rv-insights/electron`: `0.0.96`

## 实现范围

- `agentRuntimeRunnerV2` 默认策略从“仅 env 显式 `1` 开启”调整为“默认开启，显式关闭回滚”。
- 显式关闭值：`0`、`false`、`off`、`no`、`disabled`。
- 显式开启值：`1`、`true`、`on`、`yes`、`enabled`。
- 空值或未知值按新默认处理为开启。
- `agentRuntimePipelineRunnerV2` 仍保持 env 显式 `1` 才启用。
- `agentRuntimeChannelsV2` 仍保持 env 显式 `1` 才启用。
