# CodeInsights 20s Intro Video Design

## Style Prompt

一个深色、精密、可审计的技术介绍片。画面像本地桌面 Agent 控制台与架构图谱的结合：低亮度墨绿黑底、清晰的信息层级、真实项目架构图嵌入玻璃化窗口、少量品牌紫和青绿作为运行状态与事件流动线。整体克制，不做营销页式夸张英雄图，重点让观众在 20 秒内理解 CodeInsights 的本地优先、Pipeline、Agent Runtime 和可扩展工具链。

## Colors

- Background: `#071015`
- Panel: `#0d1a1f`
- Foreground: `#f3f7f4`
- Muted text: `#a9b8b2`
- Brand purple: `#8f55ff`
- Runtime teal: `#20d3a2`
- Warm signal: `#f2c56b`

## Typography

- Main UI / Chinese text: `JetBrains Mono`, system sans-serif fallback
- Data / runtime labels: `JetBrains Mono`, monospace fallback
- Headlines use 800-900 weight; body copy stays 300-400 weight for dark-background readability.

## Motion

- Medium-energy technical transitions: incoming scene clip-wipes plus short scan bars.
- No jump cuts; every scene has entrance choreography.
- Ambient movement is slow, finite, and secondary to the content.

## What NOT to Do

- 不使用通用蓝紫满屏渐变或大面积霓虹。
- 不把架构图当作纯装饰，必须配合具体功能点。
- 不使用难读的小字或密集说明段落。
- 不做不可验证的功能夸张，只表达 README 和当前代码架构中已有能力。
